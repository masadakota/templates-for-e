# CallFlow

CallFlow is a web application designed to create and manipulate flow diagrams. This project is built without any frameworks or libraries, ensuring a lightweight and customizable solution for users.

## Project Structure

```
CallFlow
├── src
│   ├── index.html          # Main HTML document
│   ├── styles              # Contains CSS styles
│   │   └── main.css
│   ├── scripts             # JavaScript files
│   │   └── main.js
│   ├── components          # Reusable components
│   │   ├── flowDiagram.js
│   │   └── flowControls.js
│   └── utils               # Utility functions
│       └── index.js
├── tests                   # Unit tests
│   └── unit
│       └── main.test.js
├── .gitignore              # Git ignore file
├── package.json            # NPM configuration file
└── README.md               # Project documentation
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Navigate to the project directory:
   ```
   cd /d/projects/web/call-flow
   ```
3. Open `index.html` in your web browser to view the application.

## Usage

- The application allows users to create flow diagrams and control their execution.
- Use the controls provided in the interface to start, stop, or reset the flow.
